# tools

리눅스에서 유용하게 쓸 수 있는 패키지, 스크립트, 빌드 도구를 정리하고 커밋해 두는 저장소입니다.

이 저장소는 다음 목적을 기준으로 운영합니다.

- 리눅스 작업에 실제로 도움이 되는 도구를 모아 둡니다.
- 재현 가능한 형태로 스크립트와 설정을 남깁니다.
- 나중에 다시 써도 바로 이해할 수 있도록 사용법을 함께 기록합니다.
- 배포판이나 환경에 따라 필요한 빌드/다운로드 보조 스크립트를 축적합니다.

## 포함 내용

현재 포함된 항목:

- `down_srpm.sh`: Oracle Linux SRPM을 찾아 다운로드하고, SPEC 확인 및 `rpmbuild -bp` 기반 소스 추출/빌드를 돕는 스크립트

## 방향

앞으로 이 저장소에는 아래와 같은 것들을 추가할 수 있습니다.

- 패키지 다운로드/빌드 보조 스크립트
- 운영 자동화용 쉘 스크립트
- 문제 해결용 유틸리티
- 배포판별 설치/호환성 메모

## 사용 예시

```bash
./down_srpm.sh kexec-tools
./down_srpm.sh kexec-tools --changelog CVE
./down_srpm.sh kexec-tools --no-cache
```

## 메모

공개 저장소로 관리할 예정이므로, 가능한 한 범용적으로 쓸 수 있는 내용 위주로 정리합니다.
